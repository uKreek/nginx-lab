Create a web server in Docker using Nginx and connect an HTML page

## Author
Merkulova Elizaveta, AM-2

## How to start a project

1. Clone repo:
```bash
  git clone https://github.com/uKreek/nginx-lab
  cd nginx-lab
```
2. Run the container:
```bash
  docker-compose up -d --build
```
3. Open in browser:
```http://localhost:8080```
